# Country Codes

Generates a Java enumeration from CSV containing different country codes in ISO 3166.

## Generated values

* ISO3166-1-Alpha-2
* ISO3166-1-Alpha-3
* ISO3166-1-numeric

## Data source

https://datahub.io/core/country-codes
